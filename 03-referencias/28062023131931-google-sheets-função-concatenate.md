# Google Sheets função CONCATENATE

- A função CONCATENATE concatena strings
- Sintaxe:
```
=CONCATENATE(string1, [string2, ...])
```
- `string1`: é a string inicial.
- `[string2...]`: representa as strings que serão acrescentadas à concatenação.
